## 比较和交换
数组元素在 [0, n-1] 范围内，可以将值为 i 的元素放到第 i 个位置上。
排序之后的数字i将出现在下标为i的位置上，可以根据下标定位对应的元素。
基于比较和交换来是实现。

time:O(N) space:O(1)

 ## 辅助空间  
numbers[i] 元素的范围在 0~n-1  
使用boolean[] 保存对应的数是否出现，索引对应着key(数的值)，值对应着value(数的特性-是否存在)

思路 ：第一次出现置为 true, 第二次根据第一次置的值或者初始的值进行判断

时间：O(n), 空间：O(n)
