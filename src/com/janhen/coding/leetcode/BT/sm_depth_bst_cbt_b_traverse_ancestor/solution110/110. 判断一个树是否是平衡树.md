**判断一个树是否是平衡树**
[110. Balanced Binary Tree(easy)](https://leetcode.com/problems/balanced-binary-tree/)

```
Given the following tree [3,9,20,null,null,15,7]:
    3
   / \
  9  20
    /  \
   15   7
Return true.

Given the following tree [1,2,2,3,3,null,null,4,4]:
       1
      / \
     2   2
    / \
   3   3
  / \
 4   4
Return false.
```
