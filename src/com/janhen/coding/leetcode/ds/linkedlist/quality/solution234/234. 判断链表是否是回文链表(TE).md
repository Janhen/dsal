[234. Palindrome Linked List(easy)](https://leetcode.com/problems/palindrome-linked-list/)
1360:208

```
Input: 1->2
Output: false

Input: 1->2->2->1
Output: true
```  
   
Follow up:
Could you do it in O(n) time and O(1) space?


## 思路1
 将左半部分的链表放入栈
 一个指针指向右半部分的第一个节点(奇数为下一个)
 两者逐个比较
 time: O(n), space: O(1)
 
 
 ## 思路2
 反转右部分链表
 之后逐个比较左半部分和右半部分
