[328. Odd Even Linked List(Medium)](https://leetcode.com/problems/odd-even-linked-list/)
599:201

链表元素按奇偶分割并拼接
奇偶指的是在链表中的位置，非值

复杂链表复制最后拆分阶段类似
```
Input: 1->2->3->4->5->NULL
Output: 1->3->5->2->4->NULL

Input: 2->1->3->5->6->4->7->NULL
Output: 2->3->6->7->1->5->4->NULL
```  

## 思路1
 2->1->3->5->6->4->7->NULL
    ↓
 2->3->5->4->7
    1->
    ↓
 2->3->6->4->7->NULL
    2->5->
    ↓
 2->3->6->7
 1->5->4->NULL
    ↓
 2->3->6->7->1->5->4->NULL
 将链表查分成两个按照奇偶个数划分的链表
 合并两条链表
