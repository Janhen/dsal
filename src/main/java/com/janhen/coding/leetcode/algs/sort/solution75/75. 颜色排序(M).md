```html
Input: [2,0,2,1,1,0]

```
 
## 思路1
带有重复元素的快速选择算法


定义多个区间  
- [0...lt] < v  
- [lt+1,i-1] = v
- [i] iterate
- [i+1,gt-1] to be iterate
- [gt...n-1] > v
