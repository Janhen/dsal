**可不经过根节点的路径中最长的都是相同数的长度**
[687. Longest Univalue Path(Easy)](https://leetcode.com/problems/longest-univalue-path/)

```
Input:

              5
             / \
            4   5
           / \   \
          1   1   5
Output:

2

Input:

              1
             / \
            4   5
           / \   \
          4   4   5
Output:

2
```
