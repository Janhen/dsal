//148. Sort List
//https://leetcode.com/problems/sort-list/
//Medium  1128:61

```
Sort a linked list in O(n log n) time using constant space complexity.

Example 1:

Input: 4->2->1->3
Output: 1->2->3->4
Example 2:

Input: -1->5->3->4->0
Output: -1->0->3->4->5
```

## 思路1
 自底向上的归并排序
 无需递归，只需迭代
 没有通过数组索引直接获取元素
