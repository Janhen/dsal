## 思路1
链表 1 和链表 2 成环求出，  
链表 1 结束时，向链表 2 遍历  
链表 2 结束时，向链表 1 遍历  

两者相交处即为公共的交点  

 数学性质: 人为构造成环
设 A 的长度为 a + c，B 的长度为 b + c，其中 c 为尾部公共部分长度，可知 a + c + b = b + c + a。

当访问链表 A 的指针访问到链表尾部时，令它从链表 B 的头部重新开始访问链表 B；
 同样地，当访问链表 B 的指针访问到链表尾部时，令它从链表 A 的头部重新开始访问链表 A。
 这样就能控制访问 A 和 B 两个链表的指针能同时访问到交点。
