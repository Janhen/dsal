```html
在一个二维数组中（每个一维数组的长度相同），
每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。
请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。
```

## 思路1
 分析：从右上角开始找，满足二分查找性质。
 数据结构：使用row、col来记录起始的行列号，
 row  代表当前遍历的行，自减则缩小了一行的取值，col同理。
> time:O(N+M) space:O(1)
